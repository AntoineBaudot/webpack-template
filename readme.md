# Webpack Template

## Features

- Basic files support (HTML, CSS, Stylus, Images, Fonts)
- Extracted CSS
- Transpilled JS (With Babel)
- Sourcemaps
- Clean dist
- Dev server (with local network support)
- Static Fils
- Cache breaker

## Instruction

Install dependencies 

```npm i ```

Launch dev server 

```npm run dev```

Build for production 

```npm run build```
